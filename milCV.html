Mijn Game in dit project

in dit project heb ik een game genaamd space invaders.
space invaders is een game waar je in de ruimte (space) bent wat natuurlijk in de naam zit.
verder heb jij de controle over een ruimteschip en is de bedoeling om de ufo's neer te schieten.
als dit jou succesvol lukt heb je gewonnen en kan je het opnieuw doen totdat je er klaar mee bent.
als je verliest mag je het opnieuw proberen en komt er een pop up op het scherm dat je verloren hebt.
